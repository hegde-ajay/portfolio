# Ajay Hegde - Personal Portfolio

A modern, responsive, and beautifully designed personal portfolio website showcasing my professional experience, technical skills, and key projects as a **Site Reliability Engineer (SRE) / DevOps Engineer**.

## 🚀 Live Preview
Run a local server to view the portfolio:
```bash
npx serve .
```
Then navigate to `http://localhost:3000` in your browser.

## 🎨 Design & Features
- **Premium Dark Theme:** Built with a cohesive cyan/violet color palette and glassmorphism UI elements.
- **Dynamic Background:** Subtle grid pattern with animated glowing orbs for depth.
- **Interactive Elements:**
  - Auto-typing effect for dynamic role highlights.
  - Animated stat counters.
  - Scroll-reveal animations for sections.
  - Parallax orbs responding to mouse movement.
- **Responsive Layout:** fully optimized for desktop, tablet, and mobile devices with a custom hamburger menu.

## 🛠️ Built With
- **HTML5:** Semantic structure and accessibility.
- **CSS3:** Custom properties (variables), Flexbox/Grid layouts, CSS animations, and media queries. No external UI frameworks used.
- **Vanilla JavaScript:** DOM manipulation, scroll observers (IntersectionObserver), and custom animations.
